How it works

  • Full audit trail: every lead tracked with send timestamp in Google Sheets

04 · 4-workflow system · ~37 nodes
AI Task Coordinator
Local AI classifies incoming tasks, routes them to the right handler, tracks state at every step, and auto-recovers from failures every 30 minutes.
n8nWebhookBaserowOllamaResend
$990 / mo
est. value saved
Webhook + 30 min
How it works
Task Intake (4A): webhook → validate → log as "queued" → return 202 immediately
Task Processor (4B): AI classifies task type → routes to handler branch
Execute task → send output via email → mark "complete"
On failure: log state, send alert. Step-level tracking, not just "it broke"
Recovery Engine (4C): fires every 30 min → retries all failed tasks automatically
Talking points
  • Four coordinated workflows. This is architecture, not a single automation
  • AI brain runs locally. Zero per-classification cost, zero cloud dependency
  • Step-level state: you always know exactly where a task is and what happened
  • Failed tasks retried automatically within 30 minutes. Zero manual recovery
  • Zero silent failures. Every exception surfaces as an alert with full context
Error-handling pattern (Recovery Engine)
